# Mark Dasuki myRetail

# Tools and Technologies used
1. Java 11
2. Spring Boot
3. MongoDB
4. Gradle
5. JUnit
6. IntelliJ IDEA
7. Windows

# Setup
1. Install Java 11
    - https://www.oracle.com/technetwork/java/javase/downloads/jdk11-downloads-5066655.html
2. Install MongoDB
    - https://docs.mongodb.com/manual/installation/
3. Install Gradle
    - https://gradle.org/install/

# Start Application
```
gradle bootRun
```

# Run tests
```
gradle clean test
```
